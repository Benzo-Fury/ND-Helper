# ND Helper
ND Helper is the official bot for the [Neo Develops Discord server](https://discord.com/invite/sSVDuawYpx). 

# Primary Commands
- `/ping`
- Time: 
  - `/time set {time}`
  - `/time get {user?}`
  - `/time delete`
- `/silent-ping {user}`
